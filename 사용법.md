# randomseat 사용법

엑셀 파일을 넣으면 좌석을 랜덤 배치하고 결과를 이미지로 저장하는 프로그램입니다.

---

## 설치

```bash
pip install -r requirements.txt
```

---

## 엑셀 파일 구조

### 시트 1: 좌석 배치

| 구분            | 설명                                                               |
| --------------- | ------------------------------------------------------------------ |
| **O 표시**      | 인원을 배치할 좌석                                                 |
| **이름**        | 고정 좌석 (해당 인원이 그 자리에 고정, 명단에 있어도 재배치 안 함) |
| **빈칸/흰색**   | 미사용 좌석 (이미지에 표시만 됨)                                   |
| **회색(solid)** | 통로 등 좌석 아님                                                  |
| **교단**        | 교단 영역 (합병 셀 지원)                                           |

- 테두리 안 흰색 셀 = 좌석 영역
- 회색 셀 = 좌석 아님 (통로 등)
- 아래에서 3번째 행과 교단 사이 = 회색(비좌석)

### 시트 2: 참가자 명단

| 컬럼     | 설명                                   |
| -------- | -------------------------------------- |
| **이름** | 참가자 이름 (`이름`, `name` 등 지원)   |
| **기수** | 기수 (`기수`, `gisu`, `batch` 등 지원) |

첫 행은 헤더로 사용됩니다.

---

## 실행

```bash
python3 main.py seat.xlsx -o result.png
```

### 옵션

| 옵션             | 설명                                             |
| ---------------- | ------------------------------------------------ |
| `-o`, `--output` | 출력 이미지 경로 (기본: `배치결과_날짜시간.png`) |
| `--no-history`   | 이번 배치를 기록에 저장하지 않음                 |
| `--seed N`       | 랜덤 시드 (같은 결과 재현용)                     |

### 예시

```bash
python3 main.py seat.xlsx                    # 기본 파일명으로 저장
python3 main.py seat.xlsx -o 1주차.png       # 지정 파일명으로 저장
python3 main.py seat.xlsx --no-history       # 기록 저장 안 함
python3 main.py seat.xlsx --seed 42          # 시드 고정
```

---

## 배치 규칙

1. **같은 기수끼리 인접하지 않도록** 분산 배치
2. **앞/뒤 편향 없이** 골고루 배치
3. **과거 배치 기록**을 반영해 이전에 앉았던 자리를 피해 배치

---

## 결과물

- **이미지 파일**: 좌석 배치 결과 (PNG)
- **배치 기록**: `seat_history.json`에 저장 (다음 배치 시 반영)

---

## 기록 초기화

테스트 후 기록을 지우고 새로 시작하려면 `seat_history.json`을 비웁니다.

```bash
echo "[]" > seat_history.json
```
